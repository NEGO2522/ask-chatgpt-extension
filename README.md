# Ask ChatGPT — Chrome Extension

Select any text on any webpage, right-click, and instantly send it to ChatGPT.

## How it works
1. Select any text on any webpage
2. Right-click on the selected text
3. Click **"Ask ChatGPT"**
4. ChatGPT opens automatically with your text — ready to answer!

## Features
- Works on any website
- Text is automatically sent to ChatGPT
- No copy-pasting needed
- Lightweight and fast
- No data collected

## Privacy
This extension does not collect, store, or transmit any user data.
Selected text is only used to open ChatGPT in a new tab.
